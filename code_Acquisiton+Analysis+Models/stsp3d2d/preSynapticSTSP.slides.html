Biophysics of short term plasticity¶

Roberto García Medina$^1$, Marco Arieli Herrera-Valdez$^1$¶

$^1$ Laboratorio de Fisiología de Sistemas, Facultad de Ciencias, Universidad Nacional Autónoma de México¶

Let $c$ represent the calcium concentration in the presynaptic terminal. Assume that \begin{eqnarray} \partial_t c &=& \frac{c_{\infty}-c}{\tau_c} + c_{in} ,\quad \textrm{presynaptic calcium concentration} \\ \partial_t r &=& \alpha_c c^n (1-r) - \beta_c r = \frac{r_{\infty}(c)-r}{\tau_r(c)},\quad \textrm{activation of release machinery by calcium} \\ \partial_t q &=& \frac{q_{\infty}-q}{\tau_q} + \delta q r,\quad \textrm{normalized number of quanta in readily releasable pool} \end{eqnarray} The parameters $c_{\infty}$ and $q_{\infty}$ represent the steady state concentration of calcium in the presynaptic terminal and the quanta in the readily releasable pool normalized by its maximum, respectively. The parameters $\tau_c$ and $\tau_q$ represent the time constants for the recovery of the presynaptic calcium concentration and the refilling of vesicles in the readily releasable pool.

The normalized quanta of neurotransmitter released is $rq$.

Assume the activation of calcium dependent proteins in the presynaptic release machinery is instantaneous. That is, $r=r_{\infty}(c)$. Then the system reduces to \begin{eqnarray} \partial_t c &=& \frac{c_{\infty}-c}{\tau_c} + c_{in}, \\ \partial_t q &=& \frac{q_{\infty}-q}{\tau_q} + \delta q r_{\infty}(c). \end{eqnarray}

def Hill(u,c,n=2):
    uu = u**n
    return uu/(uu+ c**n)

def RK2_Autonomous(f,pars, eParNames=[], eParList=[]):
    """Second-order Runge-Kutta method to solve x' = f(x,t) with U(t[0]) = U0.
    NOTES:
        This version is based on the algorithm presented in "Numerical
        Analysis", 6th Edition, by Burden and Faires, Brooks-Cole, 1997.
    """
    U=sc.zeros((pars['nSteps'], sc.prod(sc.shape(pars['ic']))),"float64")
    U[0]=pars['ic']
    for i in range(pars['nSteps']-1):
        for k in range(len(eParNames)):
            pars[eParNames[k]]= eParList[k][i]
        #t = stepSize * i
        k1 = pars['stepSize'] * f( U[i], pars) / 2.0
        U[i+1] = U[i] + pars['stepSize'] * f( U[i] + k1, pars)
    return U.transpose()

def presynSTSP(U,p):
    c,r,q= U
    rSS = Hill(c,p['c_halfAct_r_muM'],p['c_coop_r'])
    tau_r = rSS/p['alpha_r']
    dc = (p['c_PreInfty_muM'] - c)/p['tau_c_Pre'] + p['c_PreIn']/p['stepSize']
    dr = (rSS-r)/tau_r
    dq = (p['q_Infty']-q)/p['tau_q'] - r*q/p['stepSize']
    return sc.array([dc,dr,dq])

def presynSTSP_2D(U,p):
    c,q= U
    rSS = Hill(c,p['c_halfAct_r_muM'],p['c_coop_r'])
    dc = (p['c_PreInfty_muM'] - c)/p['tau_c_Pre'] + p['c_PreIn']/p['stepSize']
    dq = (p['q_Infty']-q)/p['tau_q'] - rSS*q/p['stepSize']
    return sc.array([dc,dq])
